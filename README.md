# Electron BookManager Desktop App

This is a cross platform bookmanager desktop application built on electron

### Version
1.0.3

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To run electron

```sh
$ npm start
```

### To Package & Build

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
